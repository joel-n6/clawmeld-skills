# Research Rubric

Score outputs 1-5 on each dimension.

## 1. Question clarity
Did the output identify the real question behind the capture?

## 2. Relevance
Did the research stay tightly tied to the user's likely decision?

## 3. Concreteness
Did it provide specifics instead of generic advice?

## 4. Evidence quality
Were important external claims backed by sources?

## 5. Decision usefulness
Could the user act on the output immediately?

## 6. Brevity
Was it compact enough to be read quickly without losing utility?

## Passing bar

- No dimension below `3`
- Decision usefulness should usually be `4` or `5`
- If evidence quality is below `3`, clearly label uncertain conclusions

## Common failure modes

- summary instead of research
- too many facts with no recommendation
- vague risks
- no next steps
- sources omitted when claims depend on external validation
